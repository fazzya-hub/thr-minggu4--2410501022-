- ## Deskripsi Aplikasi
[Disini aku membuat aplikasi untuk tracker selama ramadhan jadi apa aja to do list selama ramadhan ini  untuk fiturnya terdapat checkbox untuk memastikan apakah to do list tersebut dapat digunakan atau tidak dan kita dapa input text untuk menambahkan task nyaterdapat dark mode dan light mode terdapat detail screen juga untuk melihat status apa sudah selesai atau belum dan dapat menghapus task jika sudah selesai lalu terdapat progres trackernya juga jika sudah di done untuk to do listnya maka progresnya akan menambah dan saya menggunakan persen. Itu saya penjelasan simple tentang aplikasi halal tracker]

## Hooks yang Digunakan
- useState: [dipakai di dashboardscreen.js,taskcontext.js,themecontext.js, untuk menyimpan, status false atau true ]
- useEffect: [dipakai di taskcontext.js, themecontext.js. digunakan untuk mengambil data saat memulai dan menyimpan data saat ada perubahan]
- useReducer: [ADD_TASK, TOGGLE_DONE,DELETE_TASK,HYDRATE_DATA]
- Custom Hook: [useTaskHandler & fungsinya itu sebagai perantara]
